## Project

Smart Home Energy Monitoring System

## Objective

The ThingSpeak cloud platform is used to store, visualize, and analyze energy monitoring data generated by the ESP32/Python simulation.

---

## Step 1: Create ThingSpeak Account

1. Visit https://thingspeak.com
2. Create a free account.
3. Verify email and login.

---

## Step 2: Create a New Channel

Channel Name:

Smart Home Energy Monitoring

Description:

IoT-based Smart Home Energy Monitoring System using ESP32, ThingSpeak, and Python. The system monitors voltage, current, power consumption, energy usage, electricity cost, and overload conditions in real time.

---

## Step 3: Configure Fields

| Field   | Name    |
| ------- | ------- |
| Field 1 | Voltage |
| Field 2 | Current |
| Field 3 | Power   |
| Field 4 | Energy  |
| Field 5 | Cost    |
| Field 6 | Alert   |

Save the channel.

---

## Step 4: Obtain API Keys

Navigate to:

Channel → API Keys

Copy:

* Write API Key
* Read API Key

Example:

Write API Key:

YKRO5WKSVGGEWJSK

---

## Step 5: Configure Python Application

In main.py:

```python
CHANNEL_API_KEY = "YOUR_WRITE_API_KEY"
```

Data is uploaded using:

```python
requests.post(
    "https://api.thingspeak.com/update",
    data=payload
)
```

## Step 6: Create Dashboard Widgets

### Voltage Gauge

Type: Gauge

Field: Voltage

Range: 200 – 250 V

---

### Current Gauge

Type: Gauge

Field: Current

Range: 0 – 10 A

---

### Power Gauge

Type: Gauge

Field: Power

Range: 0 – 2500 W

---

### Energy Display

Type: Numeric Display

Field: Energy

Units: kWh

---

### Cost Display

Type: Numeric Display

Field: Cost

Units: INR (₹)

---

### Overload Alert

Type: Lamp Indicator

Field: Alert

0 = Normal

1 = Overload

Color: Red

---

## Step 7: Run Simulation

Execute:

```bash
python main.py
```

The application uploads data every 15 seconds to ThingSpeak.
